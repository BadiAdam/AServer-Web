Windows PC & Android Platformları İçin

Unified Minecraft
Server Hosting.

Karmaşık kodlar, modem ayarları veya Java kurulumlarıyla uğraşmayın. AServer ile tam fonksiyonlu PaperMC ve Fabric sunucularınızı Windows veya Android cihazınızda tek tıkla barındırın.

Windows için İndir v1.0.0 Resmi .msi Yükleyicisi
Android için İndir Resmi .apk Dosyası

Çapraz Platform Gücü.

AServer for PC

Masaüstü gücünü kullanın. Resmi Windows Yükleyicisi (.msi) ile kurun. Dahili Java motoru, Taskkill kalkanı ve tam telemetri radarı ile premium bir deneyim.

AServer for Mobile

Sunucunuzu cebinizde taşıyın. Modrinth entegrasyonu, akıllı mod mağazası ve canlı konsol ile Termux veya PRoot ayarlarıyla uğraşmadan mobil barındırma.

Mühendislik Harikası Özellikler.

Otomatik Java Motoru

Seçtiğiniz Minecraft sürümüne (Vanilla, Paper, Fabric) göre gerekli olan Java versiyonunu (8, 17, 21) sistem otomatik olarak bulur, indirir ve gömer. Bilgisayarınıza Java kurmanıza gerek kalmaz.

Playit.gg Entegrasyonu

Modemden port açma (Port Forwarding) devri bitti. Dahili Playit entegrasyonu ile tek tıkla sunucunuzu tüm dünyaya güvenli bir tünel üzerinden açın.

Canlı Telemetri

Sunucunuzun tükettiği RAM'i ve sisteminizin genel CPU yükünü canlı grafiklerle izleyin.

Anti-Zombi Kalkanı

Sunucu kapandığında arkada sızıntı yapan Java işlemleri bırakmaz. Görev yöneticisini temiz tutar.

Akıllı Konsol

Renkli loglar, RCON komut desteği ve oyuncu banlama/whitelist arayüzü.
